개요

사용 형식
  1. inline 방식 :
    태그 내에 코드를 직접 작성하는 방식 (간단한 소스 코드일 경우)
  2. internal 방식 :
    script 태그를 선언하여 내부에
    소스코드를 작성하는 방식
  3. external 방식 :
    외부의 .js 파일을 가져와 현재 웹 문서에 등록하는 방식
    (like css를 link 태그로 가져오는 방식)
작성 시 명심할 사항
소스코드는 함수(≒ 메소드) 단위로 작성하고, 이벤트 속성을 통해 실행한다.
이벤트 속성은 태그마다 지원하는 종류가 다를 수 있으며
일반적으로 on이벤트명 = 실행할 함수명([전달인자 | 매개변수]) 으로
이벤트와 스크립트를 연결(binding) 시킨다.

inline : 태그에 직접 코드 삽입

internal : 태그를 활용하여 코드 연결

external : 외부의 JS파일에 명시한 코드 연결
