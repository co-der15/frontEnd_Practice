정규표현식

정규 표현식(Regular Expression : Regex | RegExp)
특정한 규칙을 가진 문자열을 '검색'할 때 사용하는 형식 언어
** 형식 언어 : 스스로 프로그래밍을 할 수 없으나 특정 형식을 제공하여
소스 코드 작성에 도움을 주는 언어
** 자바 | SQL | JS | Python 등 각종 프로그래밍 언어에서 대부분 이 기능을
제공하고 있기 때문에, 한 번 익혀두면 쓸 일이 반드시 있다!

정규 표현식 객체 생성

대체 문자

정규 표현식을 사용하면 문자열 객체를 replace() 함수 사용 시에
대체 문자열로 사용할 수 있다.
** 대신맨 : 특정 문자열을 대체해서 다른 문자열로 바꾸는 것
** 이 때에 원본 문자열은 바뀌지 않는다.

플래그 문자

사용형식 : /찾을 문자열/[flag][flag . . . ]
g : global / 문자열 전체에서 일치하는 값을 여러 개 찾겠다
i : insensitive / 영문자의 경우 대 소문자를 가리지 않겠다!
m : multi / 한 줄이 아닌 여러 줄에서 문자열을 찾겠다

앵커 문자

찾고자 하는 문자열의 시작과 끝을 기준으로 잡는 문자열 검색 형식
^ : 처음 시작하는 문자
$ : 마지막에 끝나는 문자
ex )
^ABC : ABC로 시작하는 문자열
ABC$ : 문장이 끝날 때 ABC로 끝나는 문자열

메타문자

정규표현식의 핵심기능!
특정 범위의 규칙을 가진 문자를 찾을 때 사용한다.

. : 임의의 문자 하나를 찾는 형식

메타 문자를 활용한 주민번호 조회

주민번호(RRN) :    

추가 메타 문자

\d : 숫자 아무거나 하나
\w : 아무 문자나 한 문자(숫자 포함)
\s : 공백문자(탭, 띄어쓰기, 줄 바꿈)
\D : 숫자가 아닌 문자 하나
\W : 특수문자들 중 하나
\S : 공백이 아닌 문자 하나

주민번호(RRN) :    

수량 문자

a+ : a가 적어도 1 개 이상 반복
a* : a가 0 혹은 여러 개
a? : a가 0 혹은 1
a{숫자} : a를 숫자 번 반복하겠다.
a{2,5} : a가 2 ~ 5까지 반복
a{5,} : a는 5개 이상
a{,10} : a가 많아도 10개 까지
