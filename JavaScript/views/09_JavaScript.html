배열 기본 모양

자바에서는 배열을 선언할 때 특정 자료형 별로 선언이
가능했지만, 자바스크립트는 자료형을 별도로 선언하지 않기 때문에
마치 자바의 list처럼 각종 자료형을 하나의 배열에 담을 수 있다.

배열 선언

배열 함수

indexOf('값')

지역 :   

concat() : 두 개의 배열을 하나로 합칠 때 사용

join : 배열을 하나의 문자열로 합치는 함수

sort() : 배열을 오름차순으로 정렬하는 함수

shift() : 배열 맨 앞의 요소 제거
unshift() : 배열 맨 앞에 새로운 요소 추가

slice() : 배열의 특정 부분을 잘라내는 함수
splice() : 배열의 특정 인덱스를 자르고,
원하는 값을 추가할 때 사용

toString() 함수 : 배열 --> 문자열
