선택자 1

선택자란?
특정한 HTML 태그 요소를 선택할 때 사용하는 표기 방식
원하는 태그를 '선택'하여 원하는 스타일을 적용할 수 있다.

모든 요소 선택자 : *

HTML 안의 모든 요소를 선택할 때 사용하는 기호

  • 테스트1
  • 테스트2
  • 테스트3

태그 선택자 : 태그명

HTML 안의 특정 태그를 모두 찾아 스타일을 적용하는 방법

클래스 선택자 : .클래스명

HTML 안의 특정 클래스 속성이 포함된 태그들을 찾아
스타일을 적용하는 방법

  • 햇님반 김철수
  • 달님반 배철수
  • 햇님반 김탁구
  • 달님반 오구탁
  • 풀잎반 청정원
        내용 1, 2, 3

아이디 선택자 : #

HTML 문서 내에 오직 하나뿐인 id 속성을 찾아
스타일을 적용하는 방법

  1. 선택자 a1
  2. 선택자 a2
  3. 선택자 a3
  4. 선택자 a4
  5. 선택자 a5
