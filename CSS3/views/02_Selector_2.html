속성 선택자

기본 선택자 뒤에 [ ]를 붙여
특정 속성과 해당 속성의 값을 가지는
태그를 선택하는 선택자 

        선택자[속성=속성값] { . . . } 
        = : 해당 속성값을 가질 때
        ~= : 특정 단어를 속성값으로 가질 때 (띄어쓰기로 구분)
        |= : 특정 단어를 속성값으로 가질 때 ("" 로 구분)
        ^= : 속성값이 특정 단어로 시작할 때
        $= : 속성값이 특정 단어로 끝날 때
        *= : 속성값이 특정 단어를 포함할 때

자손 선택자와 후손 선택자

자손선택자 : 현재 선택자 바로 하위에 있는 요소
후손선택자 : 현재 선택한 요소 하위의 모든 요소 

        자손 : 선택자>자손선택자{ ... }
         ->>  ul>li { color : red; }
        후손 : 선택자 후손선택자{ ... }
         ->>  html div { background : wheat; }

나는 자손 홍길동입니다.

나도 자손 홍길서입니다.

    나는 자손 리스트입니다.
  • 나는 ul의 자손이자, div의 후손입니다.

    나는 ul의 후손입니다.

  • 나는 ul의 자손이자, div의 후손입니다.
  • 나는 ul의 자손이자, div의 후손입니다.
