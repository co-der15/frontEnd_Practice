폼 입력 관련 태그

form 태그는 html 웹 문서에서 사용자가 입력하는
특정 양식의 데이터를 지정한 페이지로 전송하는 역할을 수행한다.

form 안의 input 태그들로 양식을 받을 형태를 구성하고,
해당 input 태그들은 사용자가 입력한 정보를 받아서
서버로 전달하는 역할을 수행한다.

** 속성 **
action : 작성한 데이터를 보낼 페이지 주소
method : 전달할 전송 방식을 GET / POST 와 같은
HTTP 통신의 전송 방식으로 설정한다.

input 태그의 값을 사용자에게 입력받아
서버로( 특정 페이지로) 전달할 수 있다.
이 때 데이터 전송 버튼으로 'submit' 타입을 사용한다.

HTML 태그 속성 중
id는 현재 HTML 문서 내의 고유값 (id가 겹치면 제일 첫번째만 id로 인식한다)

안의 값을 꺼내올 key(변수명)의 역할

숫자 관련 input 태그들

type = "number"

입력창에 스핀형태로 표현되나, 웹 브라우저에 따라서 슬라이드 형태로 표현 되는 경우도 있다.

type = "range"

슬라이드 바를 통해 숫자를 지정할 수 있다.

날짜 관련 input 태그들

type = "date | month | week | time | datetime-local"
