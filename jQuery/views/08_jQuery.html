태그 요소 삽입 함수

삽입 함수 1

$(A).appendTo(B) : A -> B 안의 마지막에 추가
$(A).prependTo(B) : A -> B 안의 첫번째로 추가
$(A).insertAfter(B) : A를 B 뒤에 추가(동위 요소)
$(A).insertBefore(B) : A를 B 앞에 추가(동위 요소)

삽입 함수 2

$(A).append(B) : A <- B : B를 A 안의 마지막 요소에 추가
$(A).prepend(B) : A <- B : B를 A 안의 첫번째 요소로 추가
$(A).after(B) : A <- B : B를 A 다음에 추가(동위요소)
$(A).before(B) : A <- B : B를 A 앞에 추가

요소 복제 / 삭제

clone() : 특정 요소를 복제
remove() / detach() / empty() : 요소 제거
