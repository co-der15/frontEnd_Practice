마우스 연결 이벤트

mouseenter : 해당 요소에 마우스를 올렸을 때
mouseleave : 해당 요소로부터 마우스가 떠났을 때
mouseout : 특정 요소에서 벗어나면 발생
mouseover : 마우스가 특정 요소에 올라오면 발생

hover() : 마우스 접근 시 이벤트를 동작시키는 함수

키보드 연결 이벤트

keydown : 키가 눌렸을 때
keypress : 키가 눌리는 동안
keyup : 키에서 손을 떼었을 때
